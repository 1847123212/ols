Introduction

The alternative OLS-client provides a software client for the Open Bench Logic Sniffer logic analyser hardware.

The Open Bench Logic Sniffer is the result of a collaboration between Gadget Factory and Dangerous Prototypes. They created a cheap custom implementation of the SUMP client (developed by Michael Poppitz), and released it as Open Source project.
The original SUMP project already contained a workable client of its own. While this client was feature complete and working quite well, it was no longer actively developed and maintained. Being a developer and loving Open Source, I immediately started hacking and tweaking it. The result? An alternative SUMP client.

Features

The alternative Java client provides the following features:

  • Cross platform: the client runs on Mac OSX (64-bit), Windows (32/64 bit) and Linux (32/64 bit);
  • Simple installation: no longer fiddling with the serial libraries (RXTX) in order to get the client up and running. The client embeds the suitable serial libraries for several operating systems;
  • Pluggable: adding new functionality on the fly is possible and as easy as copying files to a single directory;
  • Look good, and feel good: the client has a good look and feel, aiming at being as usable as possible.
