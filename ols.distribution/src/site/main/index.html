Logic Sniffer

An alternative OLS client

Table of contents

Introduction

The alternative OLS-client provides a client for the open source Open Bench Logic Sniffer logic analyser developed and maintained by Dangerous Prototypes.

The guys of Dangerous Prototypes created a cheap custom implementation that is compatible to the original SUMP client (developed by Michael Poppitz).
While the original SUMP client was feature complete and working quite well, it was no longer actively developed. Being Open Source, I immediately started hacking and tweaking it. The result? An alternative SUMP client.

Features

The alternative Java client provides the following features:

  • Cross platform: the client runs on Mac OSX (64-bit), Windows (32/64 bit) and Linux (32/64 bit);
  • Simple installation: no longer fiddling with the serial libraries (RXTX) in order to get the client up and running. The client embeds the suitable serial libraries for several operating systems;
  • Pluggable: adding new functionality on the fly is possible and as easy as copying files to a single directory;
  • Look good, and feel good: the client has a good look and feel and aims at being as usable as possible.
